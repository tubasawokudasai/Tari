# Tari - Smart Clipboard Manager

Tari is a simple and efficient clipboard manager for macOS.

## Features

- Automatically record clipboard history (text, images, files)
- Quick search and preview content
- Double-click to paste, support drag-and-drop sorting
- Intelligent deduplication to avoid duplicate records
- Persistent storage, safe and reliable

## Tech Stack

- Swift 5.0+ + SwiftUI
- Core Data persistence
- macOS 10.15+
