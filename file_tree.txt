# Tari 项目文件树

## 项目根目录
- Tari/              # 主源码目录
  - Assets.xcassets/     # 资源文件
    - AccentColor.colorset/
    - AppIcon.appiconset/
  - Managers/            # 管理器类
    - ClipboardDataStore.swift
    - ClipboardManager.swift
    - ImageDownsampler.swift
  - Models/              # 数据模型
    - ClipboardItem.swift
  - TariModel.xcdatamodeld/  # Core Data 模型
    - TariModel.xcdatamodel/
  - Utils/               # 工具类
    - AppIconProvider.swift
    - CryptoUtils.swift
    - Formatters.swift
  - Views/               # 视图组件
    - EffectViews.swift
    - ItemCard.swift
    - PreviewDialog.swift
  - AppDelegate.swift    # 应用委托
  - ContentView.swift    # 主内容视图
  - Info.plist
  - Persistence.swift    # 数据持久化
  - PreviewWindowManager.swift  # 预览窗口管理器
  - TariApp.swift    # 应用入口
- Tari.xcodeproj/    # Xcode 项目配置
  - project.pbxproj
- .gitignore         # Git 忽略配置
- README.md          # 项目说明文档
