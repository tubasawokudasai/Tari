# Tari 项目文件树

## 项目根目录
- Tari/              # 主源码目录
  - Assets.xcassets/     # 资源文件
    - AccentColor.colorset/
    - AppIcon.appiconset/
  - Managers/            # 管理器类
    - ClipboardManager.swift
  - Models/              # 数据模型
    - ClipboardItem.swift
  - TariModel.xcdatamodeld/  # Core Data 模型
    - TariModel.xcdatamodel/
  - Utils/               # 工具类
    - Formatters.swift
  - Views/               # 视图组件
    - EffectViews.swift
    - ItemCard.swift
  - AppDelegate.swift    # 应用委托
  - ContentView.swift    # 主内容视图
  - Persistence.swift    # 数据持久化
  - TariApp.swift    # 应用入口
- Tari.xcodeproj/    # Xcode 项目配置
  - project.xcworkspace/
  - xcuserdata/
  - project.pbxproj
- build/                 # 构建输出目录（已忽略详细内容）
